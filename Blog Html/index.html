- O que é Html?

HyperText Markup Language ou HTML, é a linguagem1 padrão para a criação e estruturação de páginas da web. Ela fornece uma estrutura básica para o conteúdo da página, permitindo que os desenvolvedores definam e organizem o layout e o design dos elementos na página. HTML é composto por uma série de tags (etiquetas) que são usadas para marcar diferentes partes do conteúdo da página, como títulos, parágrafos, imagens, links e muito mais. Essas tags são interpretadas pelos navegadores da web para renderizar o conteúdo da página conforme especificado pelo desenvolvedor. A estrutura básica de uma página HTML inclui uma tag < html> que envolve todo o conteúdo da página, uma tag < head> que contém metadados sobre a página, como título, e uma tag < body> que contém o conteúdo visível da página, como texto, imagens e outros elementos.

Cada elemento é marcado com a tag apropriada e pode conter atributos para fornecer informações adicionais, como o endereço de uma imagem ou o URL de um link. HTML é a base fundamental de qualquer página da web e é essencial para qualquer desenvolvedor web entender e dominar.

1HTML, ou HyperText Markup Language, é mais precisamente descrito como uma "linguagem de marcação", não uma "linguagem de programação". Isso porque HTML é usado para estruturar e organizar o conteúdo em uma página da web, mas não possui capacidade para realizar cálculos ou interações complexas. Em vez disso, ele define a estrutura e a apresentação do conteúdo, que é então interpretado pelos navegadores da web para serem exibidos visualmente aos usuários.

- Vs Code e extensões;

O Visual Studio Code (VS Code) é um editor de código-fonte desenvolvido pela Microsoft e é uma das ferramentas mais populares entre desenvolvedores web e programadores de software em geral. Ele é gratuito, de código aberto e está disponível para Windows, macOS e Linux. O VS Code é conhecido por sua interface intuitiva e leve, além de sua ampla gama de recursos e extensões que o tornam altamente personalizável e adequado para uma variedade de linguagens de programação e tecnologias. Algumas das características notáveis do VS Code incluem:

  1. Interface Amigável:

    O VS Code possui uma interface de usuário simples e intuitiva, com uma barra de ferramentas e menus que facilitam a navegação e o acesso às funcionalidades.

  2. Suporte a Múltiplas Linguagens:

    O editor oferece suporte a uma variedade de linguagens de programação, incluindo JavaScript, HTML, CSS, Python, Java, C#, entre outras. Ele também oferece destaque de sintaxe, formatação de código e sugestões inteligentes de código para diversas linguagens.

  3. Extensibilidade:

    Uma das maiores vantagens do VS Code é sua extensibilidade. Existem milhares de extensões disponíveis na Visual Studio Code Marketplace que adicionam funcionalidades adicionais, como suporte a linguagens específicas, integração com sistemas de controle de versão, ferramentas de depuração avançadas, entre outros.

A extensão Live Server é uma das mais populares entre os desenvolvedores web. Ela permite que você crie um servidor local em tempo real para o desenvolvimento web. Com o Live Server, você pode visualizar instantaneamente as alterações que faz no código HTML, CSS e JavaScript em um navegador da web, sem a necessidade de atualizar manualmente a página a cada alteração. Isso torna o processo de desenvolvimento mais eficiente e produtivo, pois você pode ver o resultado das alterações quase que instantaneamente enquanto trabalha no código. Em resumo, o Visual Studio Code é um editor de código altamente eficiente e personalizável, e extensões populares como o Live Server são exemplos de como ele pode ser estendido para atender às necessidades específicas dos desenvolvedores.

- Inspetor de elementos;

O Inspetor de Elementos é uma ferramenta poderosa e fundamental presente nos navegadores da web que permite aos desenvolvedores visualizar e modificar o código HTML, CSS e JavaScript de uma página da web em tempo real. Ele é uma ferramenta essencial para o desenvolvimento web, pois oferece uma maneira conveniente de inspecionar e depurar o código de uma página da web diretamente no navegador.

Aqui estão algumas das principais funcionalidades e usos do Inspetor de Elementos:

  1. Visualização e Inspeção do Código:

    O Inspetor de Elementos permite que você visualize a estrutura completa de uma página da web, incluindo todos os elementos HTML, seus estilos CSS aplicados e quaisquer scripts JavaScript que estejam em execução. Você pode navegar facilmente pelo código, expandindo e recolhendo elementos conforme necessário.

  2. Edição ao Vivo:

    Uma das características mais poderosas do Inspetor de Elementos é a capacidade de editar o código HTML e CSS ao vivo diretamente no navegador. Isso permite que você faça ajustes e experimente diferentes estilos e layouts em tempo real, sem a necessidade de modificar diretamente os arquivos fonte do seu projeto.

  3. Depuração de Estilos e Layouts:

    Com o Inspetor de Elementos, você pode identificar e corrigir problemas de estilo e layout em uma página da web. Você pode inspecionar as regras CSS aplicadas a cada elemento, visualizar as dimensões e posições dos elementos na página e fazer ajustes conforme necessário para garantir que a aparência da página seja a desejada em diferentes dispositivos e tamanhos de tela.

  4. Depuração de JavaScript:

    Além de inspecionar e modificar o HTML e CSS, o Inspetor de Elementos também pode ser usado para depurar scripts JavaScript. Você pode visualizar mensagens de console, breakpoints, variáveis e a pilha de chamadas do JavaScript em execução na página, ajudando a identificar e corrigir problemas de lógica ou comportamento do código.

  5. Análise de Desempenho:

    O Inspetor de Elementos também oferece ferramentas para analisar o desempenho de uma página da web, incluindo informações sobre o tempo de carregamento, o uso de recursos e quaisquer gargalos que possam estar impactando a velocidade e responsividade da página.

Exemplo Prático:

  1. Abra o inspetor de códigos na página.
  2. Clique com o botão direito no elemento que deseja editar e selecione "Inspecionar".
  3. Edite o HTML ou CSS diretamente no inspetor para ver as mudanças ao vivo na página.

- Tags e atributos básicos do HTML;

As tags HTML são a base para criar e estruturar páginas web. Elas indicam ao navegador como exibir o conteúdo da página. As tags são cercadas por sinais de menor ( < ) e maior (> ), e muitas vêm em pares: uma tag de abertura e uma tag de fechamento.

Tags de Títulos

<h1> a <h6>: Essas tags são usadas para definir títulos e subtítulos em uma página web. <h1> representa o título mais importante e é geralmente usado para o título principal da página. As tags <h2> a <h6> são usadas para subtítulos de diferentes níveis de importância, com <h6> sendo o menos importante. Usar essas tags ajuda a estruturar o conteúdo e melhora a acessibilidade e o SEO.

Tag de Parágrafo

<p>: Define um parágrafo de texto. É a tag mais comum para agrupar e separar blocos de texto. Parágrafos são importantes para organizar o conteúdo e facilitar a leitura.

Tags de Estilização de Texto

<mark>: Destaca o texto, geralmente com um fundo amarelo, como se fosse marcado com um marcador de texto. É útil para chamar a atenção para partes específicas do texto.

<small>: Exibe o texto em um tamanho menor do que o texto ao seu redor. É frequentemente usado para informações adicionais ou disclaimers.

<i>: Aplica itálico ao texto. Tradicionalmente usado para destacar palavras estrangeiras, nomes de obras de arte, ou enfatizar uma palavra ou frase.

<u>: Sublinha o texto. Pode ser usado para dar ênfase visual, embora o uso de sublinhado em links seja mais comum.

<strong>: Indica que o texto é de grande importância, aplicando negrito ao texto. É semântico, significando que transmite um sentido de urgência ou ênfase.

Tags de Lista

<ol>: Cria uma lista ordenada, onde os itens são numerados. Usada quando a ordem dos itens é importante.

<ul>: Cria uma lista não ordenada, onde os itens são marcados com pontos ou outros marcadores. Usada para listas onde a ordem não é importante.

<li>: Define um item em uma lista. Usada dentro de <ol> ou <ul> para enumerar itens.

Tag de Link

<a>: Cria um hyperlink, que pode ser usado para navegar para outra página web, um ponto específico na mesma página ou para iniciar uma ação, como enviar um e-mail. O atributo href especifica o destino do link.

Tag de Linha Horizontal

<hr>: Insere uma linha horizontal na página, que pode ser usada para separar seções de conteúdo. Visualmente divide conteúdo sem adicionar texto.

Tags de Subscrito e Sobrescrito

<sub>: Exibe texto subscrito, ou seja, abaixo da linha base do texto normal. Usada para fórmulas químicas ou matemáticas.

<sup>: Exibe texto sobrescrito, ou seja, acima da linha base do texto normal. Usada para expoentes em fórmulas matemáticas e notas de rodapé.

Tag de Bloco de Citação

<blockquote>: Define uma citação em bloco. Geralmente é usada para citar trechos de texto mais longos que vêm de outra fonte. É apresentada com um recuo em relação ao texto ao redor.

Tag de Texto Removido

<del>: Indica que o texto foi removido ou é irrelevante. Normalmente exibido como texto riscado. Pode ser usado para mostrar edições em textos, como ao indicar que uma frase foi removida.

Tag de Abreviação

<abbr>: Define uma abreviação ou acrônimo. O atributo title pode ser usado para fornecer a explicação completa do que a abreviação significa, ajudando na acessibilidade e clareza.

- Links e ancoragem;

Links e Ancoragens no HTML

Tag de Link: <a>

A tag <a> é usada para criar hyperlinks. Um hyperlink pode levar o usuário a outra página web, um ponto específico na mesma página ou iniciar uma ação como enviar um e-mail. A tag <a> é uma das mais versáteis e frequentemente utilizadas em HTML.

Atributos Principais da Tag <a>
  • href: Define o destino do link. Pode ser um URL absoluto (como https://www.example.com) ou um URL relativo (como page2.html).
  • target: Define onde o link será aberto. Os valores comuns incluem:
    • _self (abre no mesmo frame ou janela, é o padrão)
    • _blank (abre em uma nova janela ou aba)
    • _parent (abre no frame pai)
    • _top (abre no topo do frame)
  • title: Adiciona uma dica ao link, que aparece quando o usuário passa o mouse sobre o link.
Exemplos de Uso
  • Link para Outra Página Web: <a href="https://www.example.com">Visite o Example</a>
  • Link para Outra Página do Mesmo Site: <a href="page2.html">Ir para a Página 2</a>
  • Abrir Link em uma Nova Aba: <a href="https://www.example.com" target="_blank">Visite o Example</a>
  • Adicionar um Tooltip ao Link: <a href="https://www.example.com" title="Clique para visitar o Example">Visite o Example</a>

Ancoragem com a Tag <a>

Ancoragens são usadas para criar pontos de referência dentro de uma página, permitindo que o usuário navegue rapidamente para essas seções.

Criando uma Âncora
  • Definir a Âncora: Usar a tag <a> com o atributo id para definir um ponto de ancoragem. Exemplo: <a id="section1"></a>
  • Linkar para a Âncora: Usar o atributo href com um # seguido do valor do id da âncora. Exemplo: <a href="#section1">Ir para a Seção 1</a>
Exemplos de Uso
  • Definindo uma Âncora e Linkando para Ela:
    • <h2 id="section1">Seção 1</h2>
    • <p>Este é o conteúdo da Seção 1.</p>
    • <a href="#section1">Voltar para a Seção 1</a>
  • Navegar para um Ponto Específico na Mesma Página:
    • <p><a href="#footer">Ir para o Rodapé</a></p>
    • <p>Conteúdo longo...</p>
    • <footer id="footer"> <p>Este é o rodapé da página.</p> </footer>
